# Podote

`Podote` : 포도(Grapes) + Todolist + Note

[podote](https://podote.com)

> ## commit message
>
> `feat` : 기능 관련  
> `chore` : package.json( npm 모듈, npm script등 ), configuration settings 관련  
> `style` : css 관련, file(img,svg,png등) 관련  
> `docs` : 문서 관련  
> `refactor` : 코드 리팩토링 관련 (단순 코드 포매팅, 성능 최적화등)  
> `types`: Type 수정  
> `test` : 테스트 관련  
> `fix` : 버그 수정 관련  
> `comment` : 주석 관련
